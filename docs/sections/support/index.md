# Support

## Contact Us!
If you are planning to contribute, need help setting up AIoD at your institute, or have any questions, please contact [Cameron Shand](https://www.crick.ac.uk/research/find-a-researcher/cameron-shand).

Note that we endeavour to constantly improve AIoD, but as noted on our [roadmap](../development/index.md#roadmap) we have limited resources and so direct priority accordingly. Although we will assist all with AIoD, as a core facility our external support has limited bandwidth!

## Raising an Issue
The best way to raise a specific issue would be to raise an issue on GitHub in the [relevant repository](../../index.md#repositories).

Please follow [general good practices](https://dev.to/usooldatascience/writing-great-github-issues-for-open-source-projects-a-newbies-guide-2c4d) and the specific guidelines/template of the specific repository!
