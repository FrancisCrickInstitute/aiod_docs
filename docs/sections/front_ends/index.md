# User Interfaces

As the computation of AIoD happens within the Nextflow pipeline, we can create any user interface (UI) as a front-end for users.

This could be anything, such as:

- A plugin in an imaging software (Napari, ImageJ, QuPath etc.)
- A web interface
- A standalone GUI

At present, we have a [Napari plugin](./napari_plugin/index.md) implemented. If you would like to create a plugin/UI, then please see our [Contributing](../contributing/front_ends.md) section.
